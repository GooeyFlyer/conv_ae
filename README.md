# conv_ae

A convolutional autoencoder designed for anomaly detection in multivariate time-series data.

## Usage
Move a *.csv* file into the data folder.

Open *configuration.yml* and change your file_path.

You can also change any other settings you wish.

Run **conv_ae.py**

Anomalies are listed in *anomaly_stats.txt*

Reconstructed graphs and stats of the model are saved in *images/*
